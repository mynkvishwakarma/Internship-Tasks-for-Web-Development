Web Development Basics

Learn HTML, CSS, and JavaScript fundamentals

1. HTML - The Structure of Web Pages

HTML (HyperText Markup Language) is the standard markup language for creating web pages. It describes the structure of a web page using elements and tags.

Basic HTML Elements

  • <h1> to <h6> - Headings
  • <p> - Paragraphs
  • <a> - Links
  • <img> - Images
  • <div> - Content division
  • <button> - Clickable button

Simple HTML Example

<!DOCTYPE html> <html> <head> <title>My Page</title> </head> <body> <h1>Welcome to My Website</h1> <p>This is a paragraph.</p> <a href="https://example.com">Visit Example.com</a> </body> </html>

2. CSS - Styling Web Pages

CSS (Cascading Style Sheets) is used to style and layout web pages. It lets you change colors, fonts, spacing, and more.

Common CSS Properties

  • color - Text color
  • background-color - Background color
  • font-family - Text font
  • padding - Space inside element
  • margin - Space outside element
  • border - Element border

Simple CSS Example

body { font-family: Arial, sans-serif; background-color: #f0f0f0; margin: 0; padding: 20px; } h1 { color: navy; text-align: center; } .button { background-color: #4a8eff; color: white; padding: 10px; border-radius: 5px; }

3. JavaScript - Adding Interactivity

JavaScript is a programming language that makes web pages interactive. It can respond to user actions, modify content, and much more.

What JavaScript Can Do

  • Change HTML content
  • Change HTML attributes
  • Change CSS styles
  • Respond to user events
  • Show alert messages
  • Validate form input

Simple JavaScript Example

// Display alert when button is clicked document.getElementById("myButton").addEventListener("click", function() { alert("Button was clicked!"); }); // Change text when page loads window.onload = function() { document.getElementById("welcome").innerHTML = "Welcome!"; };
